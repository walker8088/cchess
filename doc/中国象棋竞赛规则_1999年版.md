
中国象棋竞赛规则（1999年版）

## 第一章 行棋规定

### 第1条 棋盘和棋子

**1.1** 象棋盘由九道直线和十道横线交叉组成。棋盘上共有**九十个交叉点**，象棋子就摆在和活动在这些交叉点上。

- 棋盘中间没有划通直线的地方，叫做"**河界**"
- 划有斜交叉线的地方，叫做"**九宫**"
- 九道直线：红方从右到左用中文数字一至九表示；黑方用阿拉伯数字1至9表示

**1.2** 棋子共有三十二个，分为红、黑两组，每组十六个，各分七种：

| 红方 | 数量 | 黑方 | 数量 |
|------|------|------|------|
| 帅 | 1 | 将 | 1 |
| 车 | 2 | 车 | 2 |
| 马 | 2 | 马 | 2 |
| 炮 | 2 | 炮 | 2 |
| 相 | 2 | 象 | 2 |
| 士 | 2 | 士 | 2 |
| 兵 | 5 | 卒 | 5 |

**1.3** 对局开始前，双方棋子在棋盘上的摆法见标准开局图（印刷体棋图规定：红方在下用阳文；黑方在上用阴文）。

**1.4** 标准比赛用棋盘规格：
- 每格正方形，长宽均应为 **3.2至4.6cm**
- 圆形棋子直径应为 **2.7至3.2cm**
- 棋盘和棋子底色为白色或浅色
- 直线和横线为红色或深色
- 棋子面色分红黑两组，字体和圆框醒目

演示比赛用的大棋盘为直式，红方在下黑方在上。

### 第2条 走棋和吃子

**2.1** 对局时由执**红棋的一方先走**，双方轮流各走一着，直至分出胜、负、和。

轮到走棋的一方将某个棋子从一个交叉点走到另一个交叉点，或吃掉对方棋子占领其位置，都算走了一着。双方各走一着称为一个**回合**。

**2.2 各种棋子的走法**：

| 棋子 | 走法说明 |
|------|---------|
| 帅(将) | 每着一着只许走一步，前进、后退、横走均可，但不能走出"九宫"。将和帅不准在同一直线上直接对面 |
| 士(仕) | 每着一着只许沿"九宫"斜线走一步，可进可退 |
| 相(象) | 不能越过"河界"，每着斜走两步（俗称"相走田字"）。田字中心有别的棋子时（俗称"塞象眼"），不许走过去 |
| 马 | 每着走一直(或一横)一斜（俗称"马走日字"）。要去的方向有棋子挡住（俗称"蹩马腿"）不许走 |
| 车 | 每着可以直进、直退、横走，不限步数 |
| 炮(不吃子时) | 走法同车一样 |
| 兵(卒)(未过河) | 每着只许向前直走一步 |
| 兵(卒)(已过河) | 每着可向前直走或横走一步，但不能后退 |

**2.3 吃子规则**：
- 走一着棋时，己方棋子能走到的位置有对方棋子存在即可吃掉并占领
- **炮吃子必须隔一个棋子跳吃**（俗称"炮打隔子"）
- 除帅(将)外其他棋子都可以听任对方吃或主动送吃
- 吃子一方必须立即把被吃掉的棋子拿走

### 第3条 将死和困毙

**3.1** 一方的棋子攻击对方的帅(将)，并在下一着要把它吃掉，称为"**照将**"，简称"**将**"。（不必声明）

被"照将"的一方必须立即"**应将**"。如果无法应将，就算被"**将死**"。

**3.2** 轮到走棋的一方无子可走，就算被"**困毙**"。

### 第4条 胜、负、和

**4.1 输棋(负)的情况**：

| 条款 | 说明 |
|------|------|
| 4.1.1 | 帅(将)被对方"将死" |
| 4.1.2 | 走棋后形成帅(将)直接对面 |
| 4.1.3 | 被"困毙" |
| 4.1.4 | 在规定时限内未走满规定的着数 |
| 4.1.5 | 超过比赛规定的迟到判负时限 |
| 4.1.6 | 走棋违反行棋规定 |
| 4.1.7 | 走棋违反禁例，应变着而不变 |
| 4.1.8 | 在同一棋局中三次"犯规" |
| 4.1.9 | 自己宣布认输 |
| 4.1.10 | 拒绝遵守规则或严重违反纪律 |

**4.2 和棋的情况**：

| 条款 | 说明 |
|------|------|
| 4.2.1 | 双方均无可能取胜的简单局势 |
| 4.2.2 | 一方提议作和，另一方同意 |
| 4.2.3 | 双方循环反复三次，符合"不变作和"规定 |
| 4.2.4 | 符合自然限着——连续60个回合双方都没吃过子（可酌减） |

## 第五章 裁判细则

### 第24条 待判局面的裁定

双方着法循环反复三次一方要求裁决即为待判局面：

| 情形 | 裁决 |
|------|------|
| 24.1 一方长将 | **立即判负** |
| 24.2 双方均为允许/禁止着法 | **不变作和** |
| 24.3 一方禁止+一方允许 | **禁方变着，不变判负** |

## 第六章 棋例总纲

### 第25条 棋例释义

**25.1** 对局中有时出现双方着法循环不变的重复局面。据以裁处这种局面的规则条例，称为"**棋例**"。

**25.2 术语定义**：

| 类别 | 包含内容 |
|------|---------|
| **禁止着法** | 长将、长杀、长捉、一将一杀、一将一捉、一杀一捉等循环攻击手段 |
| **允许着法** | 闲着(含兑、献、拦、跟)、数将一闲、数杀一闲、数捉一闲等 |

**25.3 子力价值说明**：
- 算"子力"的：车、马、炮、过河兵(卒)、士、相(象)
- 不算"子力"的：帅(将)、未过河兵(卒)
- 一车 ≈ 双马 ≈ 双炮 ≈ 一马一炮；马炮相等；士相相等
- 过河兵价值浮动：一兵换取数子或一子换取数兵均不算得子

### 第26条 棋例总则

| 总则编号 | 内容 |
|---------|------|
| **26.1** | 在任何情况下均不允许单方面长将 |
| **26.2** | 双方均为允许着法 → 双方不变作和 |
| **26.3** | 双方均为禁止着法（不含长将）→ 不变作和 |
| **26.4** | 一方为禁止着法另一方为允许着法 → 前者变着，不变判负 |

### 第27条 棋例通则

**27.1** 允许帅(将)本身步步叫吃对方棋子按闲着处理。其他棋子和帅(将)同时捉吃或借帅(将)之力捉吃均按捉处理。
**27.2** 兵(卒)本身捉吃不含将杀按闲处理；形成相互长打则按捉处理。
**27.3** 占据防守要点立即构成简明和棋，附带产生的捉士相(象)按闲处理。
**27.4** 凡走子兼具多种作用时从重称呼和裁处（如杀兼捉按杀，捉兼况按捉）。

### 第28条 术语解释

| 术语 | 定义 |
|------|------|
| **28.1 将(照将)** | 走子直接攻击对方帅(将)者 |
| **28.2 杀(杀着)** | 走子企图在下一着或连续照将将死对方者 |
| **28.3 捉** | 走子后能造成下一着吃掉对方无根子者 |
| **28.4 打** | 将、杀、捉等攻击手段统称 |
| **28.5 兑** | 走子可与同等子力互换吃去者 |
| **28.6 献** | 无根子送吃而对方一旦吃后不致被将死或遭受损失者 |
| **28.7 拦** | 走子阻拦对方活动而不具攻击作用者 |
| **28.8 跟** | 走子盯牵对方有根子而不具攻击作用者 |
| **28.9 闲** | 非将杀捉的性质统称（兑、献、拦、跟属闲） |
| **28.10 长将** | 连续不停照将形成循环者 |
| **28.11 长杀** | 连续不停杀着形成循环者 |
| **28.12 长捉** | 连续追捉一子或数子形成循环者 |
| **28.13 长兑/长献/长拦/长跟** | 连续不停邀兑/送吃/阻拦/盯牵 |
| **28.14 长打对长打** | 双方循环反复步步是打 |
| **28.15 长打对非长打** | 一方步步是打中间无闲着，另一方中间有闲着 |
| **28.16 有根子/无根子** | 有己方充分保护的为有根子；反之无根子。形式上是根实际不起保护作用的为假根/少根 |

### 第29条 "捉"的概念

构成"捉子"应符合以下条件：

- **29.1** 形式：能够直接吃子 / 通过连续照将吃子 / 通过完整交换过程得子
- **29.2** 产生于刚走的这着棋（上一着尚不存在）
- **29.3** 对象：有子力价值的无根子（含假根子、少根子）
- **29.4** 后果：得子后不致被将死

### 细则1：在任何情况下，均不许可单方面长将

```
着法：
(1) 车四平五　将5平6
(2) 车五平四　将6平5
(3) 车四平五　……
```

**裁决**：尽管红方只有车能走动无从变着，但因违反**长将禁例**也应判负。（如图一）

---

### 细则2：凡是走子前不存在捉而走子后造成捉的（从没捉到捉），一律按捉处理

```
着法：
(1) 车三平四　将6平5
(2) 车四平三　将5平6
(3) 车三平四　……
```

**裁决**：红方**一将一捉**。虽是车原来就能吃马未吃，但将后捉马是从没捉到捉，仍应按捉处理。**红方不变作负**。（如图二）

---

### 细则3：凡走子后预计下一着能在子力价值上构成得子者均按捉处理

```
着法：
(1) 马二退一　车9平8
(2) 马一进二　车8平9
(3) 马二退一　……
```

**裁决**：红方退马固属一捉，进马暗含下一步卧槽将弃马得车也应算捉。**红方二捉，黑方一捉一兑 → 红方不变作负**。（如图三）

---

### 细则4：凡用作为根的子捉吃对方的子也按捉处理

```
着法：
(1) 车六平五　卒5平4
(2) 车五平六　卒4平5
(3) 车六平五　……
```

**裁决**：尽管红车吃卒要丢马仍按**红方长捉处理 → 红方不变作负**。（如图四）

---

### 细则5：凡捉子兼具兑、献、送吃也按捉处理

```
着法：
(1) 马八进六　车2平3
(2) 马六退八　车3平2
(3) 马八进六　……
```

**裁决**：红马长捉黑车，尽管存在马送卒口也按**捉处理**。黑方一捉一闲，红方则为长捉 → **红方变着，不变作负**。（如图五）

---

### 细则6：凡是原来已经捉着子的子走动后无新捉行为（即从捉到捉）均按闲着处理

**示例一（图六）**：

```
(1) 马八进七　炮1进1
(2) 马七退八　炮1退1
(3) 马八进七　……
```

→ 黑车依仗炮保护能吃底马但避炮属于**从捉到捉的闲着**。红马长捉无根黑炮违反禁例 → **红方不变作负**

**示例二（图七）**：

```
(1) 炮八平九　车2平1
(2) 炮九平八　车1平2
(3) 炮八平九　……
```

→ 红炮走动前后红车均能吃黑中象，避炮属于**从捉到捉**。黑车长捉红炮违反禁例 → **黑方作负**

**示例三（图八）**：

```
(1) 马三退一　车7平8
(2) 马一进三　车8平7
(3) 马三退一　……
(4) 马一进三　……
```

→ 红马长捉黑车违反禁例，黑方两步平车属**从捉到捉的闲着** → **红方不变作负**

### 细则7：凡形式上捉子一旦吃子立即会被对方将死者均按闲着处理

```
着法：
(1) 炮一平二　车9平8
(2) 炮二平一　车8平9
(3) 炮一平二　……
```

**裁决**：红方平炮**长杀**，黑方两步动车看似捉炮实际并不能吃（红方垫车还杀）。**二打对二闲 → 红方不变作负**。（如图九）

---

### 细则8：双方均为禁止着法（不含单方面长将）不变作和

```
着法：
(1) 车六平五　炮4平2
(2) 车五平六　炮2平4
(3) 车六平五　……
```

**裁决**：红方**一将一捉**，黑炮平2做杀平4增加黑马退6杀着。**双方二打还二打 → 不变作和**。（如图十）

---

### 细则9：其他棋子和帅(将)同时捉吃对方棋子均按捉处理

```
着法：
(1) 帅六平五　卒5平4
(2) 帅五平六　卒4平5
(3) 帅六平五　……
```

**裁决**：红帅捉卒本来是闲但双马也同时轮流捉卒按**长捉处理 → 红方不变作负。（如图十一）

---

### 细则10：过河兵(卒)子力价值浮动——主动送吃过河兵以换取他子不算得子

```
着法：
(1) 车一进四　车5平6
(2) 车一退四　车6平5
(3) 车一进四　……
```

**裁决**：红方进出车均含有主动送吃过河兵换黑马或车的企图。但**过河兵价值浮动不存在得子**故按**二闲处理 → 双方不变作和**。（如图十二）

---

### 细则11：发生互打时兵(卒)叫吃子均按捉处理

```
着法：
(1) 兵八平七　车3平2
(2) 兵七平八　车2平3
(3) 后八平七　……
```

**裁决**：红方两步平兵本为闲着，但黑方平车分捉马相形成**互捉**于是红方两步平兵也按捉处理。**二打对二打 → 双方不变作和**。（如图十三）

---

### 细则12：凡走子兼具多种作用时从重定性

```
着法：
(1) 炮三平四　车6平7
(2) 炮四平三　车7平6
(3) 炮三平四　……
```

**裁决**：红炮步步追捉黑车按**捉处理**不应因送给黑炮吃而按闲处理 → **红方不变作负**。（如图十四）

### 细则13：占据防守要点立即形成简明和棋附带产生的捉士相(象)按闲处理

**示例一（图十五）**：

```
(1) 车五平六　将4平5
(2) 车六平五　将5平4
(3) 车五平六　……
(4) 车六平五　……
```

→ 红方**一将一跟**占据守和要点附带产生的捉象按**闲着**处理；黑方**一杀一闲** → **双方不变作和**

**示例二（图十六）**：

```
(1) 帅五进一　炮4平3
(2) 帅五退一　炮3平4
(3) 帅五进一　……
(4) 帅五退一　……
```

→ 黑方炮不离位固守防御形成**理论上公认的和棋**，附带产生的捉士捉相按**闲着**处理 → **双方不变作和**
