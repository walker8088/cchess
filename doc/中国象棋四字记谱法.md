
# 中国象棋记谱法

**中国象棋四字记谱法**（也称中式记谱法）是传统最常用的记录方式，每一步棋用**四个字**（有时因区分多子会变成五个字）来表示，简洁清晰，便于阅读和复盘。

## 1. 棋盘纵线（路）编号规则

棋盘有9条纵线（竖线），**每方从自己右侧向左数**：

- **红方**（下方，先手）：用**汉字**表示 → 一、二、三、四、五、六、七、八、九  
- **黑方**（上方，后手）：用**阿拉伯数字**表示 → 1、2、3、4、5、6、7、8、9  

**注意**：红方“三路”与黑方“7路”是同一条纵线（相对关系），记录时始终以**走棋方自身视角**为准。

## 2. 四字记谱法基本格式

**第1字**：移动的棋子名称  
**第2字**：该棋子当前所在的纵线（路）数  
**第3字**：移动方向  
**第4字**：目标位置或步数

### 棋子名称（常用简写）
- 红方：帅、仕、相、**车**、**马**、**炮**、**兵**  
- 黑方：将、士、象、**车**、**马**、**炮**、**卒**  

（车、马、炮、兵/卒常省略偏旁，通用）

### 第3字：移动方向（仅三种）
- **平**：横向移动（左右平移，同一条横线，仅适用于直线行棋的棋子：车、炮、帅/将、兵/卒过河后）  
- **进**（或“上”）：向前移动（朝对方方向）  
- **退**（或“下”）：向后移动（朝己方方向）  

**斜走棋子限制**：  
马、士/仕、相/象 **只能用“进/退”**，不能用“平”（因它们斜线行走）。

**兵/卒特殊**：  
- 过河前只能“进”，不能“退”或“平”；  
- 过河后可“进”或“平”，仍不能“退”。

### 第4字：目标或步数
- 如果第3字是**平** → 第4字为**目标纵线数**（红方用汉字，黑方用阿拉伯数字）  
- 如果第3字是**进/退** →  
  - 直线棋子（车、炮、帅/将、兵/卒）：第4字为**前进/后退的步数**（格数）  
  - 斜走棋子（马、士/仕、相/象）：第4字为**落点所在的纵线数**（不是步数）

## 3. 典型例子
- **炮二平五**：红炮从二路横移到五路。  
- **马8进7**：黑马从8路向前跳到7路（落点7路）。  
- **车二进三**：红车从二路向前直走3格。  
- **象3进5**：黑象从3路向前飞到5路（落点5路）。  
- **兵七进一**：红兵从七路向前走1格。  
- **炮8进4**：黑炮从8路向前直走4格。

## 4. 特殊情况（同线多子区分）

当同一方**同一条纵线**上有两个以上相同棋子时，不能只用路数区分，需用**前/后**代替第1-2字：

- **前/后 + 棋子名 + 方向 + 目标/步数**  
  - 前炮平五：同一路上前面的炮横移到五路。  
  - 后车退三：同一路后面的车后退3格。

**兵/卒更复杂**（五子可能叠多）：  
- 同一路3个兵：用**前兵、中兵、后兵**区分。  
- 同一路4-5个兵：用**前兵、二兵、三兵、四兵、五兵**（或后兵）区分。  
- 两路以上都有多兵时（如2+2型），有时**省略“兵/卒”**，直接写“中兵平七”（中间那个兵平移到七路）。
  如前兵九平八、二卒7平6，此时可省略兵（卒），记做前九平八、二7平6，以达到都用４个汉字记谱的要求。
  
**士/象例外**：即使同线，也可凭“进/退”自然区分，不必加前/后。

## 5. 其他注意事项
- 一局棋谱通常先写**红方走法**，再写**黑方走法**（或标明回合）。  
- 有些老谱用“上/下”代替“进/退”，含义相同。  
- 吃子、将军等不额外标记，只记录走法本身。
