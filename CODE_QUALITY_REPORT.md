## 1. 项目概述

**CChess** 是一个用 Python 实现的中国象棋引擎与棋谱管理系统。项目具有以下核心功能：

- **棋盘逻辑**：完整的中国象棋规则实现，包括走法生成、局面评估
- **棋谱支持**：多种棋谱格式支持（PGN、CBF、TXT、XQF）
- **引擎接口**：UCI/UCCI 协议兼容的象棋引擎接口
- **工具套件**：棋谱分析、局面验证、性能测试等工具

**技术栈**：
- Python 3.8+
- Ruff、Pylint、pytest、coverage
- 支持 Windows/Linux/macOS 跨平台
